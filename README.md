# TERMUX APK
<b>Untuk HP ANDROID 7+ ke atas :</b><br> [disini](https://f-droid.org/repo/com.termux_118.apk)</br>

<b>HOW TO INSTALL SCRIPT :</b>
```
$ pkg update -y
$ pkg upgrade -y
$ termux-setup-storage (klik ijinkan)
$ pkg install curl -y
$ pkg install php -y
$ pkg install git -y
$ git clone https://github.com/effatastore/TRI
```

<b> GO TO RUN : </b>
```
$ cd TRI
$ php tri.php
```
